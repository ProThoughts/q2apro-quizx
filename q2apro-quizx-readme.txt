# q2apro Premium Plugin: Quizx

# Description

	Transforms your question2answer forum into a community quiz.


# Installation

    - Buy the plugin and download the ZIP file provided q2apro-quizx.zip
    - Make a full backup of your q2a database before installing the plugin.
    - Extract the folder q2apro-quizx from the ZIP file.
    - Move the folder q2apro-quizx to the qa-plugin folder of your Q2A installation.
    - Use your FTP-Client to upload the folder q2apro-quizx into the qa-plugin folder of your server.
    - Navigate to your site, go to Admin -> Plugins and check if the plugin "Q2APRO Quizx" is listed.
    - Change the settings of the plugin to your needs.
	- Add the necessary widgets to the layout, as described in the plugin options. 
    - Congratulations, the plugin has been installed and your quiz is ready to launch! 
	
	- Note: This plugin has been developed to work best with the SNOW FLAT THEME.
